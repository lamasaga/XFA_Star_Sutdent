# 五维重构深度思考：v2.2 方案建议

> **文档定位**：对用户反馈的系统性回应与新方案建议  
> **讨论主题**：分数制改革 · 年级基准 · 维度再划分 · 成长真实性

---

## 一、用户反馈的问题清单与根因分析

### 问题1：低年级分数天然显低

**现象**：K1学生基础分40，成长分最多20，显示分60，被标为"一般"甚至"需关注"。  
**根因**：100分制是**绝对尺度**，K1和K11用同一个满分标准，天然对低年级不公平。  
**本质**：我们在用一个"成年人的尺子"量"小孩"。

### 问题2：每学期均匀增长不合理

**现象**：每学期固定+3.5分，小学6年和高中2年增速一样。  
**根因**：忽视了**发展心理学规律**——人的成长不是线性的。  
**本质**：小学生一年变化巨大，高中生一年变化相对平缓。

### 问题3："在校就有分"变成了"躺平分"

**现象**：基础分只要在校就会涨，学生什么都不做也能从40涨到60。  
**根因**：基础分设计初衷是"保底鼓励"，但变成了"免费午餐"。  
**本质**：学生看不到"不努力就会落后"的压力，系统缺乏负向反馈。

### 问题4：社交→德行

**现象**："社交"听起来像"会不会交朋友"，窄化了。  
**根因**：中文语境下"社交"偏人际关系技巧，不包含品格、纪律、责任。  
**本质**：我们要衡量的不只是"会不会和人相处"，而是"是不是一个品行端正的人"。

### 问题5：学术维度过大

**现象**：学术涵盖课内成绩+竞赛+英语+研究，一个维度塞了太多东西。  
**根因**：把"学科知识""语言能力""竞技能力"混为一谈。  
**本质**：英语语言类表现和数学物理成绩是两种完全不同的能力。

### 问题6：艺术体育拆分导致雷达图"不好看"

**现象**：很少有学生艺术和体育都出色，雷达图总有两个"凹坑"。  
**根因**：五维要求学生"全面发展"，但现实中大多数人 naturally 有偏好。  
**本质**：如果五维总是暴露大多数人的短板，它就不是成长记录，而是**短板诊断书**——这会让学生沮丧，而非激励。

### 问题7：五维不能真正反应成长

**现象**：学生努力了一学期，因为某维没参加活动，分数反而掉了。  
**根因**：衰减机制过于惩罚性，且维度划分没有体现"正常校园生活自然产生数据"。  
**本质**：成长应该是"我认真上课、正常参与，五维就自然好看"，而不是"我必须在五个领域都优秀才算优秀"。

## 二、核心设计哲学转变

### 从「短板诊断」到「成长记录」

| | 旧思路（v2.0/v2.1） | 新思路（v2.2） |
|---|---|---|
| **目的** | 找出学生哪里不够好 | 记录学生在哪里成长了 |
| **满分标准** | 100分（所有人同一标准） | 200分（但随年级变化） |
| **基准增长** | 在校就有（每学期+3.5） | 无负面记录才有 |
| **维度设计** | 5个独立维度，每个都要高 | 5个维度允许有强弱，但不能有"塌陷" |
| **竞赛位置** | 作为维度加速器 | **完全剥离**，放入独立荣誉系统 |
| **呈现方式** | 雷达图暴露短板 | 成长曲线展示进步 |

### 关键原则

1. **年龄匹配**：K1的满分标准和K11不同，不在同一个赛道比
2. **参与即得分**：正常上课、正常参与活动，五维就好看
3. **负面才惩罚**：只有违纪、缺勤、拖欠等行为才会阻止增长
4. **优势可突出**：某维特别强可以很高，不强但只要参与就不会低
5. **竞赛不干扰**：竞赛是特殊成就，记录在荣誉墙，不影响五维公平性

## 三、分数体系重构：200分发展分制

### 3.1 为什么200分？

- 100分制在低年级太容易"触顶"或"触底"，刻度太粗
- 200分制给每个年级足够的区分度
- K1基准40 → K11基准200，跨度160分，每步增长有足够的颗粒度

### 3.2 年级基准分设计

```
年级     K1   K2   K3   K4   K5   K6   K7   K8   K9   K10  K11
基准     40   60   80   100  120  140  155  170  182  192  200
增长     —    +20  +20  +20  +20  +20  +15  +15  +12  +10  +8
```

**设计 rationale：**

| 阶段 | 年级 | 年增长 | 理由 |
|------|------|--------|------|
| **小学快速成长期** | K1-K6 | +20/年 | 小学生变化快，一年一个样，认知、身体、社交都在飞速发展 |
| **初中稳定发展期** | K7-K9 | +12~15/年 | 青春期，变化仍在继续但趋于稳定，学科分化开始 |
| **高中成熟巩固期** | K10-K11 | +8~10/年 | 接近成人，变化放缓，在已有基础上深化 |

**每学期增长 = 年增长 ÷ 2**

### 3.3 基准增长的条件性

**旧设计（无条件）：**
```
每学期基础分 +3.5（只要在校）
```

**新设计（有条件）：**
```
每学期基准增长 = 标准增长 × 行为系数

行为系数计算：
  无负面记录          → 1.0（获得完整增长）
  轻微负面（迟到3+次）→ 0.8（增长打8折）
  中度负面（缺勤/拖欠作业）→ 0.5（增长减半）
  严重负面（作弊/严重违纪）→ 0.0（停止增长）
```

**负面清单：**

| 负面行为 | 影响系数 | 说明 |
|----------|----------|------|
| 迟到累计3+次 | 0.9 | 轻微影响 |
| 作业拖欠累计5+次 | 0.8 | 学习态度问题 |
| 无故缺勤1+天 | 0.7 | 纪律问题 |
| 课堂严重违纪 | 0.6 | 行为问题 |
| 考试作弊 | 0.0 | 诚信底线 |
| 欺凌/暴力 | 0.0 | 品德底线 |

**正面加分（在基准之上）：**

| 正面行为 | 额外加分 | 说明 |
|----------|----------|------|
| 学期全勤 | +2 | 纪律模范 |
| 作业全交 | +2 | 态度认真 |
| 获得教师表扬3+次 | +3 | 表现突出 |
| 帮助同学（被记录） | +2 | 品德优秀 |
| 担任班级职务 | +2 | 责任担当 |

### 3.4 显示方式：发展分 vs 绝对分

**不在雷达图上显示绝对分数！** 而是显示：

```
┌─────────────────────────────────────────┐
│  学术发展：155/170（K8标准）              │
│  相当于 K11 水平的 91%                    │
│  较上学期：+8 📈                         │
│  在 K8 同龄人中：前 30%                  │
└─────────────────────────────────────────┘
```

这样K1学生看到的是"40/40（K1标准）100%"，而不是"40分很低"。

但等等，这又引入了"同龄人对比"，用户之前明确说不要和别人比...

**修正：只显示自我发展分，不显示排名**

```
┌─────────────────────────────────────────┐
│  学术发展：155分                          │
│  K8 同龄人一般水平：150分                 │ ← 可选显示，作为参考线
│  你的发展状态：良好 ✅                     │
│  较上学期：+8 📈                         │
└─────────────────────────────────────────┘
```

或者更简洁：

```
┌─────────────────────────────────────────┐
│  学术 155分（良好）                       │
│  ████████████████████░░░░ 77%            │
│  本学期增长 +8 分 📈                      │
│  正常参与课堂即可维持良好水平              │
└─────────────────────────────────────────┘
```

这里的77%是"相对于K11满分200分的百分比"，作为一个长期参考线。

### 3.5 颜色标签的年级适配

**旧设计（问题）：** K1显示分50 = "需关注"（红色）  
**新设计：** 颜色基于"该年级的发展状态"而非绝对分数

```typescript
function getLabel(score: number, grade: number): { label: string; color: string } {
  const gradeBenchmark = getGradeBenchmark(grade); // K1=40, K11=200
  const ratio = score / gradeBenchmark;
  
  if (ratio >= 1.1) return { label: "超前", color: "紫色" };      // 超过年级基准
  if (ratio >= 0.9)  return { label: "良好", color: "绿色" };      // 接近年级基准
  if (ratio >= 0.7)  return { label: "正常", color: "蓝色" };      // 略低于年级基准
  if (ratio >= 0.5)  return { label: "跟进", color: "黄色" };      // 需要关注
  return { label: "需支持", color: "红色" };                       // 明显落后
}
```

**示例：**

```
K1学生：学术 45分 / K1基准40分 = 1.125 → "超前"（紫色）
K1学生：学术 35分 / K1基准40分 = 0.875 → "良好"（绿色）
K1学生：学术 25分 / K1基准40分 = 0.625 → "跟进"（黄色）

K11学生：学术 180分 / K11基准200分 = 0.9 → "良好"（绿色）
K11学生：学术 220分 / K11基准200分 = 1.1 → "超前"（紫色）
```

这样K1学生考45分是"超前"，K11学生考180分也是"良好"——**每个年级都有自己的评价标准**。

## 四、五维重新划分：v2.2 方案

### 4.1 核心思路：让每个维度都"参与即得分"

**问题回顾**：v2.1的"文艺"和"体育"拆开后，要求学生同时在五个领域都投入，大多数人 naturally 会有1-2个弱项，雷达图不好看。

**解决思路**：
1. 每个维度的设计原则是"正常校园生活自然产生数据"
2. 不需要特殊天赋或额外投入就能拿到70%+的分数
3. 只有"完全不参与"或"负面行为"才会低分
4. 竞赛、考级等特殊成就剥离到独立荣誉系统

### 4.2 新五维方案

#### 维度1：学品（Learning）

**定义**：课内学习的品质与成效。**不包含竞赛，不包含英语语言类**。

| 构成 | 权重 | 说明 |
|------|------|------|
| 课堂参与 | 30% | 出勤、发言、互动（教师记录） |
| 作业完成 | 25% | 提交率、质量（系统自动+教师评价） |
| 知识掌握 | 30% | 期中/期末/单元测成绩 |
| 学习态度 | 15% | 教师评语标签（认真、主动、思考等） |

**为什么英语被移除？** 英语是语言能力，和数学物理是不同性质的能力，应该独立。

**为什么竞赛被移除？** 竞赛需要额外投入和天赋，不是"正常校园生活"的一部分。竞赛获奖记录在**荣誉墙**。

**正常参与能拿多少分？**
- 每节课都到、偶尔发言：课堂参与 70%
- 作业按时交、质量中等：作业 70%
- 考试成绩中等：知识掌握 60%
- 态度认真：学习态度 80%
- 综合：约 70%（对K1是28分/40，对K11是140分/200）

#### 维度2：言品（Language）

**定义**：语言表达与沟通能力，**包含母语和外语**。

| 构成 | 权重 | 说明 |
|------|------|------|
| 中文表达 | 40% | 课堂发言、演讲、写作、辩论 |
| 英文能力 | 40% | 英语课表现、口语、写作、演讲 |
| 沟通技巧 | 20% | 表达清晰度、倾听能力、非语言沟通 |

**为什么独立成维？** 语言能力是21世纪核心素养，不分中英文。一个学生数学不好但英语演讲很棒，应该在言品维度得到认可。

**正常参与能拿多少分？**
- 语文课正常参与、完成写作：中文 70%
- 英语课正常参与、口语练习：英文 65%
- 能正常交流：沟通 70%
- 综合：约 68%

#### 维度3：行品（Conduct）

**定义**：德行品行，由"社交"升级而来。包含纪律、诚信、尊重、责任、服务意识。

| 构成 | 权重 | 说明 |
|------|------|------|
| 纪律遵守 | 25% | 出勤、守时、遵守校规 |
| 诚信正直 | 25% | 作业诚信、考试诚信、言行一致 |
| 尊重友善 | 20% | 尊重师长、友善待人、包容差异 |
| 责任担当 | 20% | 完成分内事、班级职务、承认错误 |
| 服务精神 | 10% | 帮助他人、社区服务、公益活动 |

**为什么不是"社交"？** "社交"偏重人际关系技巧（会不会交朋友、会不会说话），而"行品"强调品格和道德。一个学生可能内向朋友少（社交弱），但诚信正直尊重他人（行品强）。

**正常参与能拿多少分？**
- 不迟到早退、不违纪：纪律 90%
- 不作弊、不撒谎：诚信 95%
- 尊重他人、不欺凌：尊重 90%
- 完成作业、值日认真：责任 80%
- 偶尔帮助他人：服务 60%
- 综合：约 83%（这是最容易高分的维度，因为正常学生都具备基本品德）

#### 维度4：身心（Body & Mind）

**定义**：身体健康与心理状态的**外在表现**。包含体育参与、身体素质、健康习惯、情绪管理的**可观察行为**。

| 构成 | 权重 | 说明 |
|------|------|------|
| 体育参与 | 35% | 体育课出勤、运动会参与、日常运动 |
| 身体素质 | 30% | 体测成绩、体能进步 |
| 健康习惯 | 20% | 作息规律、饮食健康、运动习惯 |
| 情绪表现 | 15% | 情绪稳定性（教师观察）、心情记录 |

**关键设计**：
- **不包含艺术/审美**（艺术拆出到独立维度）
- **心理状态不作为独立维度**，而是作为"系数"影响所有维度
- 但心理状态的**可观察外在表现**（情绪稳定性、心情记录）放在身心维度

**正常参与能拿多少分？**
- 体育课全勤、运动会参与：体育参与 80%
- 体测及格：身体素质 60%
- 作息基本规律：健康习惯 70%
- 情绪无明显问题：情绪表现 80%
- 综合：约 73%

#### 维度5：美品（Aesthetics）

**定义**：审美素养与艺术参与。**不强调专业技能，强调参与和感知**。

| 构成 | 权重 | 说明 |
|------|------|------|
| 艺术参与 | 40% | 参加艺术节、美术课、音乐课、戏剧活动 |
| 审美感知 | 30% | 参观展览、欣赏演出、文化体验（记录即得分） |
| 创作表达 | 20% | 完成课堂艺术作品、原创尝试（不要求专业水平） |
| 文化理解 | 10% | 了解传统文化、跨文化理解 |

**关键设计**：
- **不需要考级、不需要获奖**
- 参加一次校园艺术节 = 高分
- 完成一次美术课作业 = 得分
- 参观一次博物馆 = 得分
- 只有"从不参与任何艺术活动"才会低分

**正常参与能拿多少分？**
- 美术课/音乐课正常参与：艺术参与 80%
- 参观过展览/看过演出：审美感知 70%
- 完成课堂作品：创作表达 70%
- 听过传统文化讲座：文化理解 60%
- 综合：约 74%

### 4.3 五维总览

| 维度 | 核心 | 不含 | 正常参与得分 |
|------|------|------|-------------|
| **学品** | 课内学习品质 | 竞赛、英语 | ~70% |
| **言品** | 语言表达（中英文） | — | ~68% |
| **行品** | 德行品行 | — | ~83% |
| **身心** | 体育健康+情绪表现 | 艺术 | ~73% |
| **美品** | 艺术参与+审美感知 | 专业技能 | ~74% |

**五个维度正常参与的综合均值：约 74%**

这意味着：**一个正常参与校园生活的学生，五维都应该在"良好"区间。只有完全不参与或负面行为才会导致低分。**

### 4.4 竞赛和特殊成就的归属

**完全剥离出五维！**

| 成就类型 | 记录位置 | 展示位置 |
|----------|----------|----------|
| 学科竞赛获奖 | 荣誉墙 | 个人档案-荣誉 |
| 英语竞赛获奖 | 荣誉墙 | 个人档案-荣誉 |
| 艺术考级 | 里程碑 | 个人档案-技能 |
| 体育比赛获奖 | 荣誉墙 | 个人档案-荣誉 |
| 科创竞赛获奖 | 荣誉墙 | 个人档案-荣誉 |
| 发明专利 | 里程碑 | 个人档案-创新 |

**五维只记录"参与"和"过程"，不记录"竞技结果"。**

**为什么？**
- 竞赛需要天赋+额外投入+资源，不是每个学生的校园生活标配
- 如果五维包含竞赛，不参加竞赛的学生永远拿不到满分
- 竞赛成就在荣誉墙中展示，同样能获得认可

## 五、心理健康系数：从维度到系数

### 5.1 保留系数设计

心理状态不作为五维之一，但作为**影响所有维度的系数**。

```
最终显示分 = 维度原始分 × 心理健康系数

系数范围：0.85 ~ 1.10
```

**系数等级：**

| 心理状态 | 系数 | 对200分制的影响 |
|----------|------|----------------|
| 极佳 | 1.10 | +20分（200→220） |
| 良好 | 1.05 | +10分 |
| 正常 | 1.00 | 无影响 |
| 一般 | 0.92 | -16分 |
| 需关注 | 0.85 | -30分 |

**为什么保留系数？**
- 心理状态确实影响所有表现（这是科学事实）
- 但学生不需要"刷心理分"，只需要正常记录心情、有需要时求助
- 系数是系统自动计算的，不需要学生主动操作

### 5.2 职业兴趣推荐引擎

同样保留，但不影响分数，只提供发展方向建议。

## 六、雷达图展示方式改革

### 6.1 问题：传统雷达图总是暴露短板

```
        学品 85
       /    \
      /      \
美品 72 ──┼─── 言品 78
      \      /
       \    /
        身心 70
        行品 88

"我有三个凹坑，我好差" —— 学生的真实感受
```

### 6.2 新展示方式：成长轨迹为主，雷达图为辅

**主视图：五维成长曲线（时间轴）**


## 七、待讨论的关键问题

### 问题1：200分制是否太复杂？

**我的思考**：200分制对学生来说只是数字变大了，理解成本没有增加。关键是**呈现方式**——学生看到的是"155/170 良好"，不需要理解200分制的数学。

**备选**：保留100分制，但用"发展百分比"呈现。
- K1学生考40分 → 显示"100%（K1标准）"
- K11学生考80分 → 显示"40%（K11标准）"

但这样学生看到的是百分比，可能更抽象。

### 问题2："身心"合并体育和心理是否太杂？

**我的思考**：确实有点杂。但如果不合并，又会出现"体育"和"心理"两个维度，心理作为状态已经被降级为系数了。

**备选方案**：
- 方案A：保持"身心"（体育+情绪表现）
- 方案B：拆为"体育"和"心品"（但心品又接近状态了）
- 方案C：体育融入"学品"（体育课也是课），但这样体育被弱化了

我倾向方案A，因为情绪管理的**可观察外在表现**（是否经常情绪低落、是否有攻击行为）和体育健康是一体的"身心状态"。

### 问题3：美品维度会不会成为"摆拍维度"？

**我的思考**：如果设计为"参观展览即得分"，学生可能为了刷分而"打卡式参观"。

**解决**：
- 参观后需要提交一份简短反思（50字）才能得分
- 教师确认学生的参与是真实的
- 强调"感知"而非"打卡"

### 问题4：行品（德行）怎么量化？

**我的思考**：德行确实最难量化。

**解决**：
- 主要依靠**负面清单**（不扣分就是高分）
- 正面行为作为额外加分（帮助他人、承担责任）
- 教师评语标签作为辅助
- **底线思维**：正常学生只要不违纪，行品就应该在80%以上

## 八、总结：v2.2 核心改变

| 方面 | v2.1 | v2.2（建议） |
|------|------|-------------|
| **满分** | 100分 | **200分** |
| **年级基准** | 无（K1和K11同一标准） | **每个年级有自己的基准** |
| **增长速度** | 每学期固定+3.5 | **小学+20/年，初中+12/年，高中+8/年** |
| **基准增长条件** | 在校就有 | **无负面记录才有** |
| **五维** | 学术、文艺、体育、社交、创新 | **学品、言品、行品、身心、美品** |
| **竞赛** | 维度加速器 | **完全剥离，荣誉墙展示** |
| **英语** | 在学术维度内 | **独立为"言品"维度** |
| **社交** | 独立维度 | **升级为"行品"（德行）** |
| **艺术+体育** | 拆分为两个维度 | **体育在身心，艺术在美品** |
| **心理** | 系数 | **保持系数** |
| **呈现方式** | 雷达图为主 | **成长曲线为主，雷达图为辅** |
| **评价标准** | 绝对分数 | **相对于年级基准** |
