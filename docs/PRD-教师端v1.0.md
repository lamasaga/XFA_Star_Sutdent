# PRD：教师端 v1.0

> **文档定位**：教师端功能设计  
> **对应学生端**：PRD-六维成长体系v3.1-修订.md  
> **核心理念**：教师是"学生成长档案的记录者和引导者"，不是"打分机器"  
> **设计原则**：极简操作、智能辅助、数据驱动、批量优先

---

## 一、教师端定位

### 1.1 用户画像

| 角色 | 核心诉求 | 使用频率 | 操作容忍度 |
|------|----------|----------|-----------|
| **班主任** | 快速了解全班状态，高效写评语，识别需关注学生 | 每天 | 低（事务多） |
| **任课老师** | 给学生写学科评语，记录课堂表现 | 每周1-2次 | 低（时间紧） |
| **体育老师** | 录入体测数据，记录体育课表现 | 每学期2次 | 中等 |
| **心理老师** | 追踪心理状态，预警干预 | 每周 | 中等 |
| **年级组长/管理** | 查看班级对比，数据报表 | 每周 | 中等 |

### 1.2 与六维成长体系的对应关系

```
学生端：「看见自己」→ 主动经营成长
教师端：「看见学生」→ 高效记录、精准引导、及时干预
管理端：「看见全局」→ 数据驱动决策
```

教师端的核心价值：**用最少的操作，为学生成长档案提供最丰富的数据。**

#### 3.1.2 班级六维雷达图

- **展示内容**：班级六维平均分雷达图（6个维度）
- **数据口径**：该班级所有学生当前学期六维显示分的平均值
- **对比线**：年级平均（虚线）+ 班级平均（实线）
- **点击交互**：点击某维度，下钻查看该维度班级排名/分布
- **颜色标注**：
  - 班级均值 ≥ 年级均值 120% → 紫色（遥遥领先）
  - 班级均值 ≥ 年级均值 105% → 绿色（优于平均）
  - 班级均值 ≥ 年级均值 95% → 蓝色（接近平均）
  - 班级均值 ≥ 年级均值 80% → 黄色（低于平均）
  - 班级均值 < 年级均值 80% → 红色（明显落后）

#### 3.1.3 需关注学生列表

系统自动标记需要教师关注的学生：

| 关注类型 | 触发条件 | 标记颜色 | 教师行动 |
|----------|----------|----------|----------|
| **心理预警** | 心理测评风险等级≥中度 / 心情记录连续中断14天 | 🔴 红色 | 约谈/转介心理老师 |
| **偏科预警** | 单一维度低于理论满分65% / 标准差>20 | 🟠 橙色 | 针对性鼓励/推荐活动 |
| **衰减预警** | 某维度连续两学期下降 | 🟡 黄色 | 了解原因/家校沟通 |
| **纪律预警** | 迟到3+次 / 缺勤1+天 / 违纪记录 | 🔵 蓝色 | 班主任约谈 |
| **低参与预警** | 某维度成长分仅底线分（无表现/卓越） | 🟢 绿色 | 鼓励参与活动 |

列表按严重程度排序，班主任可直接点击进入学生档案。

### 模块3：评语系统（核心功能）

#### 3.3.1 设计理念

> **目标**：让教师写评语的时间从 15 分钟/人降到 3 分钟/人

**传统痛点**：
- 每学期要给 25 个学生写评语，每个 15 分钟 = 6+ 小时
- 不知道写什么，容易写成"该生表现良好"的套话
- 评语和五维分数脱节，写了评语但对分数没影响

**系统解法**：
- AI 辅助生成（关键词 → 完整评语）
- 六维标签直接关联分数
- 批量操作（一次给多人写评语）
- 模板库 + 快捷标签

#### 3.3.2 写评语流程

```
选择学生 → 填写评语文字 → 勾选六维标签 → 提交
    ↓           ↓              ↓
   AI辅助     字数提示       标签影响分数
   模板选择   情感分析       实时预览
```

#### 3.5.2 预警处理流程

```

  ├─ 已面谈 → 记录面谈摘要 → 标记为"已处理"
```

## 七、与六维成长体系的对应

| 六维维度 | 教师可录入的数据 | 数据来源 |
|----------|-----------------|---------|
| **逻辑** | 成绩、评语标签"思维活跃/逻辑清晰" | 任课老师（数理） |
| **创新** | 评语标签"有创意/动手能力强"、里程碑审核 | 科创老师 |
| **表达** | 评语标签"表达清晰/有感染力"、成绩 | 语文老师、英语老师 |
| **才情** | 评语标签"有审美力/艺术天赋"、里程碑审核 | 美术/音乐老师 |
| **身心** | 体测数据、评语标签"运动能力强/情绪稳定" | 体育老师、心理老师 |
| **德行** | 评语标签"责任心强/乐于助人/遵守纪律"、班级管理 | 所有老师、班主任 |
