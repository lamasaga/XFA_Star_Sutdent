# PRD：六维成长体系 v3.1 修订版

> **文档状态**：修订稿（基于 v3.0）  
> **修订内容**：成长分三层结构 + 德行/身心评价来源扩展  
> **核心标准**：不犯错60% · 正常表现80% · 积极参与+优秀表现100%

---

## 一、修订背景

v3.0 设计完成后，经用户反馈确认以下关键调整：

1. **成长分必须分层**：日常正常发展最多达到80%，剩余20%需要额外优秀表现
2. **德行评价来源扩展**：加上老师评价和学生互评
3. **身心评价来源扩展**：加上体育老师和心理老师的反馈
4. **明确分数标准**：不犯错→60%，正常表现→80%，积极参与+优秀表现→100%

---

## 二、成长分三层结构设计

### 2.1 核心公式

```
显示分 = 年级基准分 + 成长分

成长分 = 底线分(0~36) + 表现分(0~12) + 卓越分(0~12)
       = 总计 0~60 分

其中：
  底线分(60%)：不违纪、不缺勤、基本完成作业即可获得
  表现分(20%)：正常成绩、正常参与、正常表现
  卓越分(20%)：积极参与、优秀表现、老师好评、竞赛获奖
```

### 2.2 三层分的获取标准

| 层级 | 占比 | 分数 | 获取条件 | 学生画像 |
|------|------|------|----------|----------|
| **底线分** | 60% | 0~36 | 不违纪、不缺勤、作业基本提交 | "乖乖学生" |
| **表现分** | 20% | 0~12 | 成绩中等、正常参与课堂和活动 | "正常学生" |
| **卓越分** | 20% | 0~12 | 成绩优秀、积极参与、获奖、老师好评 | "优秀学生" |

**关键设计：底线分的36分非常容易拿到**，只要是一个守规矩的学生就能拿到。这样保证了绝大多数学生的分数不会难看。

### 2.3 各维度总分计算示例

```
以K8学生（年级基准170）为例：

学生A（只做到不违纪）：
  成长分 = 36(底线) + 0 + 0 = 36
  显示分 = 170 + 36 = 206
  相对于满分(170+60=230)：206/230 = 89.6% → "良好"

学生B（正常表现）：
  成长分 = 36(底线) + 10(表现) + 2(卓越) = 48
  显示分 = 170 + 48 = 218
  相对于满分：218/230 = 94.8% → "良好"

学生C（优秀表现）：
  成长分 = 36(底线) + 12(表现) + 12(卓越) = 60
  显示分 = 170 + 60 = 230
  相对于满分：230/230 = 100% → "超前"
```

### 2.4 颜色标签（基于"显示分/理论满分"）

```typescript
const theoryMax = 年级基准 + 60;  // 该年级该维度的理论满分
const ratio = 显示分 / theoryMax;

if (ratio >= 1.0)   return "超前"（紫色）;  // 超过理论满分（竞赛加分可能导致）
if (ratio >= 0.90)  return "优秀"（绿色）;  // 90%~100%
if (ratio >= 0.80)  return "良好"（蓝色）;  // 80%~90%（正常学生区间）
if (ratio >= 0.65)  return "正常"（黄色）;  // 65%~80%（底线学生区间）
return "需支持"（红色）;                      // <65%（明显落后）
```

**关键变化**：
- 正常学生（底线+表现 = 48分）→ 显示分/满分 ≈ 85-95% → "良好"（蓝色）
- 优秀学生（三线拉满 = 60分）→ 100% → "超前"（紫色）
- 只做到不违纪的学生（36分）→ 75-85% → "正常"（黄色）
- 有违纪行为的学生（<36分）→ <75% → "需支持"（红色）

## 三、六维详细设计（v3.1 修订版）

### 3.1 逻辑维度（Logic）

#### 成长分三层结构

```
逻辑成长分 = 底线分(0~36) + 表现分(0~12) + 卓越分(0~12)
```

**底线分(0~36) — 不违纪即可获得：**

| 行为 | 分值 | 记录方式 |
|------|------|----------|
| 数理课程（数学/物理/化学等）无迟到早退 | +12 | 出勤记录 |
| 数理课程无缺勤 | +12 | 出勤记录 |
| 数理作业基本提交（提交率≥70%） | +12 | 系统自动统计 |

**表现分(0~12) — 正常表现：**

| 行为 | 分值 | 记录方式 |
|------|------|----------|
| 数理课程成绩中等（班级排名40%-70%） | +4 | Score表 |

| 数理课程成绩优秀（班级排名前20%） | +12 | Score表 |

封顶12分。

### 4.2 学生互评机制设计

```typescript
interface PeerReview {
  id: string;
  reviewerId: string;       // 评价人
  revieweeId: string;       // 被评价人
  semester: string;
  dimensions: {
```

- 获得≥60%同学勾选即得分
- 结果只统计勾选比例，不显示具体名单

## 五、三层分数的获取难度设计

### 5.1 设计原则：让80%的学生达到"良好"

| 学生类型 | 底线分 | 表现分 | 卓越分 | 总分 | 显示分/满分 | 标签 |
|----------|--------|--------|--------|------|------------|------|
| **问题学生**（经常违纪） | 20 | 2 | 0 | 22 | 83% | 正常/需支持 |
| **底线学生**（不违纪但被动） | 36 | 2 | 0 | 38 | 89% | 良好 |
| **正常学生**（正常参与） | 36 | 10 | 2 | 48 | 95% | 良好 |
| **优秀学生**（积极参与） | 36 | 12 | 8 | 56 | 98% | 优秀 |
| **卓越学生**（全面领先） | 36 | 12 | 12 | 60 | 100%+ | 超前 |

**关键洞察：**
- 只要做到"不违纪"，就能拿到36分底线分，显示分/满分 ≈ 85-90%，标签为"良好"
- 要做到"优秀"（90%+），需要在表现分上拿到10分以上（成绩中等偏上）
- 要做到"超前"（100%），需要在卓越分上有额外表现（竞赛、获奖、老师好评）

### 5.2 偏科容忍设计

```
六维中：
  - 6个维度均 ≥ 理论满分的80% → 全面发展，无需干预
  - 5个维度 ≥ 理论满分的80% → 略有偏科，系统建议关注最低维
  - 4个维度 ≥ 理论满分的80% → 偏科明显，班主任介入
  - ≤3个维度 ≥ 理论满分的80% → 严重偏科，需要重点关注
```

## 七、总结：v3.1 核心变化

| 方面 | v3.0 | v3.1（修订） |
|------|------|-------------|
| **成长分结构** | 不分层，各类行为混算 | **三层：底线60% + 表现20% + 卓越20%** |
| **正常学生分数** | 约70-80% | **约85-95%（良好）** |
| **底线学生分数** | 不明确 | **约75-85%（正常，不违纪即可）** |
| **德行评价来源** | 教师评语+活动记录 | **+学生互评 +班主任专门评价** |
| **身心评价来源** | 出勤+体测+运动打卡 | **+体育老师反馈 +心理老师反馈** |
| **心理老师角色** | 只影响系数 | **+身心维度卓越分评语** |
| **体育老师角色** | 无直接参与 | **+身心维度表现分评定** |
| **学生互评** | 无 | **每学期匿名互评，影响德行维度** |

### 一句话概括

> **v3.1 让成长分"三层化"：底线分（不违纪就有60%）、表现分（正常成绩+参与）、卓越分（竞赛+获奖+老师好评）。德行加入学生互评和班主任评价，身心加入体育老师和心理老师反馈。正常学生的六维都在"良好"区间，只有卓越学生才能达到"超前"。**
