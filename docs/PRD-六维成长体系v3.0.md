# PRD：六维成长体系 v3.0

> **文档状态**：设计稿  
> **目标版本**：v3.0  
> **核心架构**：六维表现层 + 心理健康系数 + 竞赛里程碑系统  
> **新六维**：逻辑 · 创新 · 表达 · 才情 · 身心 · 德行

---

## 一、设计理念：参与即成长

### 1.1 核心信条

> **"学生正常参与校园生活，六维就自然生长。"**

本体系的设计目标是：
1. **降低门槛** — 不需要特殊天赋，正常上课、参加活动就能拿高分
2. **鼓励参与** — 竞赛参与即得分，获奖额外加分但不过分
3. **真实记录** — 记录学生在校的每一次尝试、每一次努力
4. **成长导向** — 和自己比，看到从入学到毕业的发展轨迹

### 1.2 与旧版本的对比

| 方面 | v2.1（旧） | v3.0（新） |
|------|-----------|-----------|
| 维度数 | 5维 | **6维** |
| 学术处理 | 一个维度塞了课内+竞赛+英语 | **拆分为逻辑+创新+表达** |
| 竞赛位置 | 维度加速器 | **里程碑系统，参与即得分** |
| 获奖加成 | 直接加在成长分上 | **额外加分不超过基础分的100%** |
| 艺术体育 | 拆分为文艺+体育 | **才情+身心，降低专业门槛** |
| 社交 | 社交维度 | **升级为德行，涵盖品格** |
| 满分 | 100分 | **200分，年级基准制** |
| 增长方式 | 每学期固定+3.5 | **有条件增长，无负面记录才有** |

## 二、分数体系：200分年级基准制

### 2.1 为什么用200分？

- 100分制在低年级太容易"触底"（K1学生正常水平只有40分）
- 200分制给每个年级足够的区分度，且与年级增长曲线匹配
- K1基准40分，K11基准200分，跨度160分，成长空间充足

### 2.2 年级基准分表

```
年级   K1   K2   K3   K4   K5   K6   K7   K8   K9   K10  K11
基准   40   60   80   100  120  140  155  170  182  192  200
年增长 —    +20  +20  +20  +20  +20  +15  +15  +12  +10  +8
```

**设计 rationale：**

| 阶段 | 年级 | 年增长 | 理由 |
|------|------|--------|------|
| **小学快速成长期** | K1-K6 | +20/年 | 认知、身体、社交飞速发展 |
| **初中稳定发展期** | K7-K9 | +12-15/年 | 青春期，学科分化开始 |
| **高中成熟巩固期** | K10-K11 | +8-10/年 | 接近成人，在已有基础上深化 |

### 2.3 每学期增长 = 年增长 ÷ 2

```
K1: +10/学期    K2: +10/学期    K3: +10/学期
K4: +10/学期    K5: +10/学期    K6: +10/学期
K7: +7.5/学期   K8: +7.5/学期   K9: +6/学期
K10: +5/学期    K11: +4/学期
```

### 2.4 基准增长的条件性

**核心规则：无负面记录，才获得完整基准增长。**

```
本学期基准增长 = 标准增长 × 行为系数

行为系数计算：
  无负面记录          → 1.0（获得完整增长）
  轻微负面（迟到3+次）→ 0.85（增长打85折）
  中度负面（缺勤/拖欠作业）→ 0.6（增长减半）
  严重负面（作弊/严重违纪）→ 0.0（停止增长）
```

**负面清单：**

| 负面行为 | 影响系数 | 说明 |
|----------|----------|------|
| 迟到累计3+次 | 0.85 | 轻微影响 |
| 作业拖欠累计5+次 | 0.7 | 学习态度问题 |
| 无故缺勤1+天 | 0.6 | 纪律问题 |
| 课堂严重违纪 | 0.5 | 行为问题 |
| 考试作弊 | 0.0 | 诚信底线，停止增长 |
| 欺凌/暴力 | 0.0 | 品德底线，停止增长 |

**正面加分（在基准之上额外获得）：**

| 正面行为 | 额外加分 | 说明 |
|----------|----------|------|
| 学期全勤 | +3 | 纪律模范 |
| 作业全交 | +3 | 态度认真 |
| 获得教师表扬3+次 | +4 | 表现突出 |
| 帮助同学（被记录） | +3 | 品德优秀 |
| 担任班级职务 | +3 | 责任担当 |

### 2.5 颜色标签（基于年级基准）

**不再用绝对分数评价！** 而是基于"相对于该年级基准"：

```typescript
function getLabel(score: number, grade: number): { label: string; color: string } {
  const benchmark = getGradeBenchmark(grade); // K1=40, K11=200
  const ratio = score / benchmark;
  
  if (ratio >= 1.15) return { label: "超前", color: "#9333ea" };  // 超过年级基准15%
  if (ratio >= 0.95) return { label: "良好", color: "#16a34a" };  // 接近或达到年级基准
  if (ratio >= 0.80) return { label: "正常", color: "#2563eb" };  // 略低于年级基准
  if (ratio >= 0.60) return { label: "跟进", color: "#ca8a04" };  // 需要关注
  return { label: "需支持", color: "#dc2626" };                    // 明显落后
}
```

**示例：**

```
K1学生：逻辑 48分 / K1基准40分 = 1.20 → "超前"（紫色）
K1学生：逻辑 38分 / K1基准40分 = 0.95 → "良好"（绿色）
K1学生：逻辑 28分 / K1基准40分 = 0.70 → "跟进"（黄色）

K11学生：逻辑 190分 / K11基准200分 = 0.95 → "良好"（绿色）
K11学生：逻辑 230分 / K11基准200分 = 1.15 → "超前"（紫色）
K11学生：逻辑 140分 / K11基准200分 = 0.70 → "跟进"（黄色）
```

**关键：每个年级都有自己的评价标准！** K1的48分和K11的190分都是"良好"。

## 三、六维详细设计

### 3.0 通用规则

**显示分公式：**
```
显示分 = (基准分 + 成长分 + 竞赛加分 + 正面加分) × 心理健康系数

其中：
  基准分 = 年级基准分（根据年级自动确定）
  成长分 = 0~60（各维度独立计算）
  竞赛加分 = 0~∞（参与即得分，获奖额外加分）
  正面加分 = 0~15（全勤、全交、表扬等）
  心理健康系数 = 0.85~1.10
```

**总封顶：** 显示分理论无上限，但系统显示时标注"相对于K11满分200分的百分比"。

### 3.1 逻辑维度（Logic）

#### 3.1.1 定义

逻辑维度衡量学生的**数理思维能力、逻辑推理能力、分析解决问题能力**。涵盖数学、物理、化学、生物、信息学等逻辑推理类学科的课内学习表现。

**不包含：** 学科竞赛（竞赛在里程碑系统中单独记录）、英语语言类。

#### 3.1.2 成长分构成（0~60）

```
逻辑成长分 = 知识掌握(0~24) + 课堂参与(0~16) + 作业质量(0~12) + 思维表现(0~8)
```

**① 知识掌握（0~24）**

| 行为 | 分值 | 记录方式 |
|------|------|----------|
| 期末考试得分率 × 12 | 0~12 | Score 表 |
| 期中考试得分率 × 8 | 0~8 | Score 表 |
| 单元测/月考得分率 × 4 | 0~4 | Score 表 |

```typescript
function calculateKnowledgeScore(scores: Score[]): number {
  const semesterScores = scores.filter(s => s.semester === currentSemester);
  const bySubject = groupBySubject(semesterScores);
  
  let weightedSum = 0;
  let totalWeight = 0;
  
  for (const score of Object.values(bySubject)) {
    const rate = score.score / (score.total || 100);
    const weight = score.examType === "FINAL" ? 3.0
                 : score.examType === "MIDTERM" ? 2.0
                 : 1.0;
    weightedSum += rate * weight;
    totalWeight += weight;
  }
  
  const avgRate = totalWeight > 0 ? weightedSum / totalWeight : 0;
  return Math.min(24, Math.round(avgRate * 24));
}
```

**② 课堂参与（0~16）**

| 行为 | 分值 | 记录方式 |
|------|------|----------|
| 教师评语"思维活跃"标签 | +3/条 | Comment.dimensions |
| 教师评语"课堂积极"标签 | +2/条 | Comment.dimensions |
| 主动回答难题/提出好问题 | +3/次 | 教师课堂记录 |
| 小组讨论中担任逻辑分析角色 | +2/次 | 教师课堂记录 |

封顶 16 分。

**③ 作业质量（0~12）**

| 行为 | 分值 | 记录方式 |
|------|------|----------|
| 作业按时提交率 ≥90% | +6 | 系统自动统计 |
| 作业质量评价「优」 | +4 | 教师评价 |
| 作业质量评价「良」 | +2 | 教师评价 |
| 作业有独特解题思路 | +3 | 教师评语标签 |

封顶 12 分。

**④ 思维表现（0~8）**

| 行为 | 分值 | 记录方式 |
|------|------|----------|
| 完成研究性课题（数理方向） | +4 | 课题记录 |
| 帮助同学讲解难题（被记录） | +2/次 | 同学反馈/教师确认 |
| 发现题目/教材中的错误 | +3 | 教师确认 |

封顶 8 分。

#### 3.1.3 正常参与能拿多少分？

```
知识掌握：成绩中等 → 得分率65% × 24 = 15.6 → 16
课堂参与：偶尔发言 → 教师评语1条 = 2
作业质量：按时交、质量良 → 6 + 2 = 8
思维表现：无 → 0

逻辑成长分 = 16 + 2 + 8 + 0 = 26
逻辑显示分 = (基准分 + 26) × 心理健康系数

以K8为例：基准170 + 26 = 196 × 1.0 = 196 / 基准170 = 1.15 → "超前"
以K8为例：基准170 + 15 = 185 × 1.0 = 185 / 基准170 = 1.09 → "良好"
```

**结论**：正常学习、成绩中等、作业按时交 = 良好水平。

#### 3.1.4 衰减机制

| 触发条件 | 衰减系数 | 说明 |
|----------|----------|------|
| 本学期无考试/测验记录 | ×0.85 | 知识掌握部分无法计算 |
| 作业长期拖欠（10+次） | ×0.70 | 学习态度严重问题 |
| 考试作弊 | ×0 | 该学期逻辑维度归零 |

### 3.2 创新维度（Innovation）

#### 3.2.1 定义

创新维度衡量学生的**创造力、动手实践能力、问题解决能力、科技素养**。涵盖科技创新、发明创造、编程、机器人、创客、研究性学习（非数理方向）等。

**与逻辑的区别：** 逻辑是"解题"，创新是"创造"。一个学生数学好（逻辑强）不一定能做出一个机器人（创新强）。

#### 3.2.2 成长分构成（0~60）

```
创新成长分 = 项目实践(0~24) + 创新参与(0~18) + 思维特质(0~12) + 成果产出(0~6)
```

**① 项目实践（0~24）**

| 行为 | 分值 | 记录方式 |
|------|------|----------|
| 完成一个完整的创新项目 | +8/个 | 项目提交记录 |
| 项目有实际应用价值 | +4 | 教师/专家评审 |
| 使用创客空间/实验室完成作品 | +3/件 | 空间使用记录 |
| 完成编程作品/小程序 | +4/件 | 学生提交 |

封顶 24 分。

**② 创新参与（0~18）**

| 行为 | 分值 | 记录方式 |
|------|------|----------|
| 参加科创社团活动 | +3/次 | Activity 表 |
| 参加创客工作坊/科技讲座 | +2/次 | Activity 表 |
| 参观科技馆/创新企业 | +2/次 | Activity 表 |
| 参与班级/学校的改进建议 | +2/条（被采纳+4） | 教师确认 |

封顶 18 分。

**③ 思维特质（0~12）**

| 行为 | 分值 | 记录方式 |
|------|------|----------|
| 教师评语"有创意"标签 | +3/条 | Comment.dimensions |
| 教师评语"善于解决问题"标签 | +3/条 | Comment.dimensions |
| 提出独特的改进方案 | +3/次 | 教师确认 |

封顶 12 分。

**④ 成果产出（0~6）**

| 行为 | 分值 | 记录方式 |
|------|------|----------|
| 获得专利/著作权 | +6 | Milestone 表 |
| 发表科创论文/报告 | +4 | Milestone 表 |
| 作品被学校/社区采用 | +3 | 教师确认 |

封顶 6 分。

#### 3.2.3 正常参与能拿多少分？

```
项目实践：无 → 0
创新参与：参加科创社团3次 = 9
思维特质：教师评语1条"有创意" = 3
成果产出：无 → 0

创新成长分 = 0 + 9 + 3 + 0 = 12
创新显示分 = (基准分 + 12) × 系数

以K8为例：基准170 + 12 = 182 / 170 = 1.07 → "良好"
```

**但等等**：如果一个学生从不参加任何科创活动，创新维度会很低。这是设计意图——**创新维度鼓励学生尝试科创活动**，但不强制（不参加活动只是该维低，不影响其他维）。

#### 3.2.4 衰减机制

| 触发条件 | 衰减系数 | 说明 |
|----------|----------|------|
| 本学期零创新活动 | ×0.80 | 创新思维需要持续锻炼 |
| 连续两学期无创新实践 | ×0.65 | 明显衰退 |

### 3.3 表达维度（Expression）

#### 3.3.1 定义

表达维度衡量学生的**语言表达能力、沟通说服力、文字功底**。涵盖中文写作、演讲辩论、英语听说读写、跨文化沟通等。

**为什么独立？** 语言能力是21世纪核心素养，中英文表达能力不应被淹没在"学术"维度中。一个学生数学不好但英语演讲很棒，应该在表达维度得到认可。

#### 3.3.2 成长分构成（0~60）

```
表达成长分 = 中文表达(0~20) + 英文能力(0~20) + 沟通实践(0~14) + 写作产出(0~6)
```

**① 中文表达（0~20）**

| 行为 | 分值 | 记录方式 |
|------|------|----------|
| 语文课表现（参与+作业） | 0~8 | 同逻辑维度的课堂参与和作业 |
| 演讲/辩论/汇报经历 | +4/次 | Activity 表 |
| 主持活动/典礼 | +5/次 | Activity 表 |
| 教师评语"表达清晰"标签 | +3/条 | Comment.dimensions |
| 教师评语"有感染力"标签 | +3/条 | Comment.dimensions |

封顶 20 分。

**② 英文能力（0~20）**

| 行为 | 分值 | 记录方式 |
|------|------|----------|
| 英语课表现（参与+作业+测试） | 0~10 | 同逻辑维度的知识掌握 |
| 英语口语展示/演讲 | +4/次 | Activity 表 |
| 英语写作作品 | +3/篇 | 学生提交 |
| 参与英语角/外语活动 | +2/次 | Activity 表 |
| 教师评语"英语优秀"标签 | +3/条 | Comment.dimensions |

封顶 20 分。

**③ 沟通实践（0~14）**

| 行为 | 分值 | 记录方式 |
|------|------|----------|
| 课堂主动发言（任何科目） | +1/次 | 教师记录（简化版） |
| 小组讨论中担任发言人 | +2/次 | 教师记录 |
| 跨文化交流活动（国际交流等） | +4/次 | Activity 表 |

封顶 14 分。

**④ 写作产出（0~6）**

| 行为 | 分值 | 记录方式 |
|------|------|----------|
| 发表文章（校刊+2, 公开刊物+4） | +2~4 | Milestone 表 |
| 完成独立写作项目（小说/诗集等） | +3 | 学生提交 |
| 博客/公众号持续更新 | +2 | 学生提交 |

封顶 6 分。

#### 3.3.3 正常参与能拿多少分？

```
中文表达：语文课正常+偶尔发言 = 4 + 2 = 6
英文能力：英语课正常+无演讲 = 5 + 0 = 5
沟通实践：偶尔发言 = 3
写作产出：无 = 0

表达成长分 = 6 + 5 + 3 + 0 = 14
表达显示分 = (基准分 + 14) × 系数

以K8为例：基准170 + 14 = 184 / 170 = 1.08 → "良好"
```

#### 3.3.4 衰减机制

| 触发条件 | 衰减系数 | 说明 |
|----------|----------|------|
| 本学期语文课/英语课缺勤5+次 | ×0.80 | 语言表达需要持续练习 |
| 长期不发言、不参与表达活动 | ×0.75 | 表达能力用进废退 |

### 3.4 才情维度（Talent）

#### 3.4.1 定义

才情维度衡量学生的**艺术感知力、审美能力、创作表达力、文化理解力**。涵盖音乐、美术、舞蹈、戏剧、书法、传统文化等。

**核心设计：不强调专业技能，强调参与和感知。**

- 参加一次校园艺术节 = 高分
- 完成一次美术课作业 = 得分
- 参观一次博物馆 = 得分
- 只有"从不参与任何艺术活动"才会低分

#### 3.4.2 成长分构成（0~60）

```
才情成长分 = 艺术参与(0~24) + 审美感知(0~18) + 创作尝试(0~12) + 文化理解(0~6)
```

**① 艺术参与（0~24）**

| 行为 | 分值 | 记录方式 |
|------|------|----------|
| 参加校园艺术节/文艺汇演 | +6/次 | Activity 表 |
| 参加美术课/音乐课/舞蹈课 | +3/次（正常出勤即得） | 出勤记录 |
| 参加文艺社团活动 | +4/次 | Activity 表 |
| 参加艺术工作坊/大师课 | +4/次 | Activity 表 |

封顶 24 分。

**② 审美感知（0~18）**

| 行为 | 分值 | 记录方式 |
|------|------|----------|
| 参观艺术展览/博物馆 | +4/次 | Activity 表 |
| 观看演出/音乐会/戏剧 | +3/次 | Activity 表 |
| 欣赏经典作品并写简短感想 | +3/次 | 学生提交（50字即可） |
| 教师评语"有审美力"标签 | +3/条 | Comment.dimensions |

封顶 18 分。

**③ 创作尝试（0~12）**

| 参加校内艺术比赛（无论获奖） | +3/次 | Activity 表 |

**④ 文化理解（0~6）**

| 行为 | 分值 | 记录方式 |
|------|------|----------|
| 学习传统文化（书法/戏曲/民俗） | +3/次 | Activity 表 |
| 了解跨文化艺术 | +2/次 | Activity 表 |
| 参与传统文化活动 | +3/次 | Activity 表 |

封顶 6 分。

#### 3.4.3 正常参与能拿多少分？

```
艺术参与：美术课全勤 = 3×16周 = 48 → 封顶24
审美感知：参观1次展览 = 4
创作尝试：完成课堂作品3件 = 9
文化理解：无 = 0

才情成长分 = 24 + 4 + 9 + 0 = 37
才情显示分 = (基准分 + 37) × 系数

以K8为例：基准170 + 37 = 207 / 170 = 1.22 → "超前"
```

**结论**：只要正常上美术/音乐课、完成课堂作品，才情维度就很容易高分。

#### 3.4.4 衰减机制

| 触发条件 | 衰减系数 | 说明 |
|----------|----------|------|
| 本学期零艺术活动（包括课程） | ×0.75 | 艺术感知不练会退化 |
| 连续两学期无艺术参与 | ×0.60 | 明显衰退 |

### 3.5 身心维度（Body & Mind）

#### 3.5.1 定义

身心维度衡量学生的**身体素质、运动习惯、健康生活方式、情绪管理的可观察外在表现**。涵盖体育课参与、运动会、日常运动、体测、作息规律、情绪稳定性等。

**核心设计：记录可观察的外在行为，不记录内在心理状态（那是系数的工作）。**

#### 3.5.2 成长分构成（0~60）

```
身心成长分 = 体育参与(0~20) + 身体素质(0~18) + 健康习惯(0~14) + 情绪表现(0~8)
```

**① 体育参与（0~20）**

| 行为 | 分值 | 记录方式 |
|------|------|----------|
| 体育课出勤 | +1/次（全学期约+16） | 出勤记录 |
| 运动会参与（无论名次） | +5/次 | Activity 表 |
| 参加体育社团/校队 | +4/次 | Activity 表 |
| 日常运动打卡 | +2/次 | 运动打卡系统 |

封顶 20 分。

**② 身体素质（0~18）**

| 行为 | 分值 | 记录方式 |
|------|------|----------|
| 体测成绩「优秀」 | +8 | 体测数据 |
| 体测成绩「良好」 | +5 | 体测数据 |
| 体测成绩「及格」 | +2 | 体测数据 |
| 体测成绩较上学期进步 | +3 | 系统对比 |
| 体育课成绩「优」 | +4 | Score 表 |
| 体育课成绩「良」 | +2 | Score 表 |

封顶 18 分。

**③ 健康习惯（0~14）**

| 行为 | 分值 | 记录方式 |
|------|------|----------|
| 每周运动 ≥3次 | +6 | 运动打卡 |
| 每周运动 1~2次 | +3 | 运动打卡 |
| 作息规律（睡眠充足） | +4 | 学生自评/家长确认 |
| 健康饮食记录 | +2 | 学生自评（可选） |

封顶 14 分。

**④ 情绪表现（0~8）**

| 行为 | 分值 | 记录方式 |
|------|------|----------|
| 心情日记连续记录14+天 | +4 | MoodEntry 表 |
| 心情日记连续记录7+天 | +2 | MoodEntry 表 |
| 教师观察"情绪稳定" | +3 | 教师评语标签 |
| 主动寻求帮助（心理咨询等） | +3 | 系统记录 |

封顶 8 分。

**注意**：情绪表现只记录**可观察的外在行为**（是否记录心情、是否主动求助），不记录内在情绪状态（那是心理健康系数的工作）。

#### 3.5.3 正常参与能拿多少分？

```
体育参与：体育课全勤 + 运动会参与 = 16 + 5 = 21 → 封顶20
身体素质：体测及格 = 2
健康习惯：每周运动2次 = 3
情绪表现：心情记录7天 = 2

身心成长分 = 20 + 2 + 3 + 2 = 27
身心显示分 = (基准分 + 27) × 系数

以K8为例：基准170 + 27 = 197 / 170 = 1.16 → "超前"
```

**结论**：体育课全勤 + 运动会参与 + 偶尔运动 = 很容易高分。

#### 3.5.4 衰减机制

| 触发条件 | 衰减系数 | 说明 |
|----------|----------|------|
| 体育课缺勤5+次 | ×0.75 | 体育参与是核心 |
| 本学期零运动记录 | ×0.70 | 身体素质下降快 |
| 连续30天无心情记录 | ×0.80 | 情绪表现需要持续关注 |

### 3.6 德行维度（Virtue）

#### 3.6.1 定义

德行维度衡量学生的**品格修养、道德行为、社会责任感、合作精神**。由"社交"升级而来，涵盖纪律遵守、诚信正直、尊重友善、责任担当、服务精神。

**核心设计：主要靠"负面清单"（不扣分就是高分），正面行为作为额外加分。**

正常学生只要不违纪，德行维度就应该在80%以上。

#### 3.6.2 成长分构成（0~60）

```
德行成长分 = 纪律遵守(0~18) + 诚信正直(0~15) + 尊重友善(0~12) + 责任担当(0~9) + 服务精神(0~6)
```

**① 纪律遵守（0~18）**

| 行为 | 分值 | 记录方式 |
|------|------|----------|
| 学期无迟到记录 | +6 | 系统自动 |
| 学期无早退记录 | +3 | 系统自动 |
| 学期无缺勤记录 | +6 | 系统自动 |
| 遵守课堂纪律（教师评价） | +3 | 教师评语 |

封顶 18 分。

**② 诚信正直（0~15）**

| 行为 | 分值 | 记录方式 |
|------|------|----------|
| 学期无作弊记录 | +6 | 系统自动 |
| 作业独立完成（无抄袭） | +5 | 教师评价 |
| 言行一致，信守承诺 | +4 | 教师评语标签 |

封顶 15 分。

**③ 尊重友善（0~12）**

| 行为 | 分值 | 记录方式 |
|------|------|----------|
| 尊重师长 | +4 | 教师评语标签 |
| 友善待人，无欺凌记录 | +5 | 教师评价 + 同学反馈 |
| 包容差异，尊重多元 | +3 | 教师评语标签 |

封顶 12 分。

**④ 责任担当（0~9）**

| 行为 | 分值 | 记录方式 |
|------|------|----------|
| 担任班级职务 | +3 | 学生干部记录 |
| 认真完成值日/集体任务 | +3 | 教师确认 |
| 承认错误并改正 | +3 | 教师确认 |

封顶 9 分。

**⑤ 服务精神（0~6）**

| 行为 | 分值 | 记录方式 |
|------|------|----------|
| 参与义工/社区服务 | +3/次 | Activity 表 |
| 帮助同学（被记录） | +2/次 | 同学反馈/教师确认 |
| 参与公益活动 | +3/次 | Activity 表 |

封顶 6 分。

#### 3.6.3 正常参与能拿多少分？

```
纪律遵守：无迟到无早退无缺勤 = 6 + 3 + 6 = 15
诚信正直：无作弊 + 独立完成作业 = 6 + 5 = 11
尊重友善：尊重师长 + 友善待人 = 4 + 5 = 9
责任担当：无职务但完成值日 = 3
服务精神：无 = 0

德行成长分 = 15 + 11 + 9 + 3 + 0 = 38
德行显示分 = (基准分 + 38) × 系数

以K8为例：基准170 + 38 = 208 / 170 = 1.22 → "超前"
```

**结论**：只要不违纪、尊重他人，德行维度就很容易高分。这是设计意图——**德行是底线，不是天花板**。

#### 3.6.4 衰减机制

| 触发条件 | 衰减系数 | 说明 |
|----------|----------|------|
| 严重违纪（欺凌/暴力） | ×0 | 该学期德行维度归零 |
| 考试作弊 | ×0.3 | 诚信严重受损 |
| 多次违纪（3+次） | ×0.5 | 行为问题 |

## 四、竞赛里程碑系统

### 4.1 设计原则

> **"参与即认可，获奖是锦上添花。"**

| 原则 | 说明 |
|------|------|
| 参与就得分 | 只要报名并参加竞赛，就有基础分 |
| 获奖额外加分 | 但额外加分不超过基础分的100% |
| 里程碑承载 | 所有竞赛记录在学生个人里程碑中 |
| 跨维度影响 | 竞赛影响的维度由竞赛类型决定 |

### 4.2 竞赛加分规则

**基础参与分：**

| 竞赛级别 | 基础参与分 |
|----------|-----------|
| 校级竞赛 | +2 |
| 区级竞赛 | +3 |
| 市级竞赛 | +4 |
| 省级竞赛 | +5 |
| 国家级/国际级 | +6 |

**获奖额外加分（不超过基础分的100%）：**

| 奖项 | 额外加分比例 | 校级示例 | 省级示例 |
|------|------------|---------|---------|
| 特等奖/第一名 | +100% | +2（总计4） | +5（总计10） |
| 一等奖/第二名 | +75% | +1.5（总计3.5） | +3.75（总计8.75） |
| 二等奖/第三名 | +50% | +1（总计3） | +2.5（总计7.5） |
| 三等奖/优秀奖 | +25% | +0.5（总计2.5） | +1.25（总计6.25） |
| 参与奖/未获奖 | +0% | +0（总计2） | +0（总计5） |

**关键规则：**
- 额外加分 **不超过基础分的100%**，即获奖最多翻倍
- 团体赛所有成员获得 **70%** 的分数（鼓励团队协作）
- 同一竞赛在多个维度获奖，取 **最高加分**（不叠加）

### 4.3 竞赛类型与维度映射

| 竞赛类型 | 示例 | 影响维度 |
|----------|------|----------|
| **数理竞赛** | 数学奥赛、物理竞赛 | 逻辑 |
| **科创竞赛** | 机器人大赛、编程竞赛 | 创新 |
| **语言竞赛** | 英语演讲、辩论赛、写作比赛 | 表达 |
| **文艺竞赛** | 钢琴比赛、绘画比赛、舞蹈比赛 | 才情 |
| **体育竞赛** | 运动会、球赛、游泳比赛 | 身心 |
| **综合竞赛** | 全能赛、综合素质评价 | 最高维度 |

### 4.4 里程碑记录

所有竞赛记录在 `Milestone` 表中：

```typescript
interface CompetitionMilestone {
  id: string;
  studentId: string;
  type: "COMPETITION";
  title: string;              // 竞赛名称
  category: "LOGIC" | "INNOVATION" | "EXPRESSION" | "TALENT" | "BODY_MIND";
  level: "SCHOOL" | "DISTRICT" | "CITY" | "PROVINCE" | "NATIONAL" | "INTERNATIONAL";
  award: "FIRST" | "SECOND" | "THIRD" | "HONORABLE" | "PARTICIPATION";
  isTeam: boolean;
  teamSize?: number;
  score: number;              // 基础参与分 + 获奖额外分
  affectedDimension: string;  // 影响的维度
  semester: string;
  occurredAt: Date;
}
```

### 4.5 竞赛示例

```
学生小明：参加省级数学竞赛获二等奖

竞赛类型：数理竞赛 → 影响维度：逻辑
竞赛级别：省级 → 基础参与分：+5
获奖等级：二等奖 → 额外加分：+50% × 5 = +2.5
总分：5 + 2.5 = 7.5

逻辑维度成长分 = 原成长分 + 7.5
（但成长分封顶60，超过部分不计算）

里程碑记录：
  "2024年省级中学生数学竞赛二等奖"
  影响维度：逻辑
  获得分数：+7.5
```

## 五、心理健康系数

### 5.1 设计定位

心理状态不作为六维之一，但作为**影响所有维度的系数**。

```
最终显示分 = (基准分 + 成长分 + 竞赛加分 + 正面加分) × 心理健康系数

系数范围：0.85 ~ 1.10
```

### 5.2 系数计算

| 心理状态 | 系数 | 对200分制的影响 |
|----------|------|----------------|
| 极佳 | 1.10 | +20分（200→220） |
| 良好 | 1.05 | +10分 |
| 正常 | 1.00 | 无影响 |
| 一般 | 0.92 | -16分 |
| 需关注 | 0.85 | -30分 |

**计算方式：** 基于心理测评（60%权重）+ 心情日记连续性（40%权重），每学期计算一次。

### 5.3 系数使用规则

- 系数是**系统自动计算**的，不需要学生主动操作
- 学生端显示"心理状态"提示（如"心理状态佳，表现有加成 💚"），但不显示具体系数
- 教师端可以看到系数及构成（测评分+心情分）
- 系数为"需关注"时，自动通知班主任和心理老师

## 六、全面发展保障机制

### 6.1 均衡度评价

```typescript
function calculateBalanceScore(dimensions: Record<string, number>): number {
  const values = Object.values(dimensions);
  const mean = values.reduce((a, b) => a + b, 0) / values.length;
  const variance = values.reduce((sum, v) => sum + Math.pow(v - mean, 2), 0) / values.length;
  const stdDev = Math.sqrt(variance);
  
  return Math.max(0, Math.round(100 - stdDev * 2));
}
```

| 均衡度 | 评价 | 说明 |
|--------|------|------|
| 90~100 | **非常均衡** | 六维齐头并进 |
| 75~89 | **较为均衡** | 有强弱之分，但整体健康 |
| 60~74 | **略有偏科** | 某个维度明显落后 |
| <60 | **偏科严重** | 需要干预和引导 |

### 6.2 偏科容忍设计

**关键改变：允许1个维度低于平均线！**

```
六维中：
  - 5个维度 ≥ 年级基准 × 0.9 → 均衡发展，无需干预
  - 4个维度 ≥ 年级基准 × 0.9 → 略有偏科，系统建议关注最低维
  - 3个维度 ≥ 年级基准 × 0.9 → 偏科严重，班主任介入
```

**理由**："人无完人"，学生在六维中有一两个弱项是正常的。系统不应该要求学生在所有领域都优秀，而是确保**没有严重塌陷**。

### 6.3 智能待办推荐

系统根据**最低维度**推荐待办：

| 最低维度 | 推荐待办1 | 推荐待办2 | 预期提升 |
|----------|-----------|-----------|----------|
| 逻辑最低 | 整理本周错题 | 找老师答疑一次 | +3~5 |
| 创新最低 | 去一次创客空间 | 参加一次科技讲座 | +4~6 |
| 表达最低 | 参加一次演讲活动 | 写一篇短文 | +4~6 |
| 才情最低 | 参观一次展览 | 完成一件课堂艺术作品 | +4~6 |
| 身心最低 | 本周运动打卡3次 | 记录本周心情 | +3~5 |
| 德行最低 | 主动帮助一位同学 | 认真完成一次值日 | +3~5 |

### 6.4 评优门槛

| 奖项 | 六维要求 |
|------|----------|
| 三好学生 | 无维度低于年级基准×0.8，均衡度 ≥ 75 |
| 优秀学生干部 | 德行 ≥ 年级基准×1.0，无维度低于0.7 |
| 全能之星 | 六维均值 ≥ 年级基准×1.0，均衡度 ≥ 85 |
| 优秀毕业生 | 无维度低于年级基准×0.9，均衡度 ≥ 80 |
| 特长之星 | 某一维度 ≥ 年级基准×1.2，其他维度 ≥ 0.7 |

## 十、验收标准

### 场景1：新六维展示

```gherkin
Given 学生查看自己的六维雷达图
```

## 十一、总结

### v3.0 核心变化

| 方面 | v2.1 | v3.0 |
|------|------|------|
| 维度数 | 5维 | **6维** |
| 维度名称 | 学术、文艺、体育、社交、创新 | **逻辑、创新、表达、才情、身心、德行** |
| 学术处理 | 一个维度含课内+竞赛+英语 | **拆分为逻辑+创新+表达** |
| 竞赛 | 维度加速器 | **里程碑系统，参与即得分，获奖额外≤100%** |
| 英语 | 在学术内 | **独立为"表达"维度** |
| 艺术 | 文艺维度 | **独立为"才情"维度，降低专业门槛** |
| 体育 | 体育维度 | **融入"身心"维度** |
| 社交 | 社交维度 | **升级为"德行"维度** |
| 满分 | 100分 | **200分，年级基准制** |
| 增长 | 每学期固定+3.5 | **有条件增长，小学快、初中中、高中慢** |
| 评价标准 | 绝对分数 | **相对于年级基准** |
| 呈现方式 | 雷达图 | **成长曲线为主，雷达图为辅** |

### 一句话概括

> **v3.0 把五维从「 mixed 状态+表现 」彻底重构为「六维纯表现 + 竞赛里程碑 + 心理健康系数」，用「年级基准制」解决低年级分数显低的问题，用「参与即得分」鼓励学生尝试各领域，用「竞赛里程碑」记录荣誉但不干扰六维公平性。**
