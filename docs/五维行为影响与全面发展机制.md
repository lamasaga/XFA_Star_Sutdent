# 五维行为影响与全面发展机制

> **文档定位**：面向学生、教师、家长的行为指南  
> **核心问题**："我在学校做什么，会影响我的五维分数？怎么做能让五个维度都变好？"

---

## 一、核心理念：做正确的事，五个维度自然开花

本系统的设计理念是：**不强制学生为了刷分而做特定行为，而是让正常参与校园生活的各种行为，自然地在五维上留下痕迹。**

学生不需要记住复杂的计分规则，只需要知道一个大原则：

> **「在学校正常学习、正常参与、正常成长，你的五维就会一起涨。长期忽视某个方面，那个维度就会提醒你。」**

### 2.2 学业维度：成绩不是唯一，但很重要

| 行为 | 影响类型 | 成长分变化 | 说明 |
|------|----------|-----------|------|
| **期中考试** | 直接加分 | +2~5 | 得分率 × 权重1.5，考得好成长分明显增加 |
| **期末考试** | 直接加分 | +3~8 | 权重2，期末考最重要 |
| **月考/单元测** | 直接加分 | +1~3 | 权重1，持续积累 |
| **作业按时完成** | 间接加分 | +0.5/周 | 通过教师评语中的态度标签影响 |
| **学科竞赛获奖** | 跨维加分 | 学业+3~5, 特长+2~3 | 竞赛同时提升学业和特长 |
| **连续缺考** | 衰减惩罚 | ×0.9 | 本学期无考试，继承上学期 × 0.9 |
| **作业长期拖欠** | 停滞 | 0 | 教师评语负面标签，成长分不涨 |

**学业维度计算公式：**
```
学业成长分 = Σ(每次考试得分率 × 考试权重) / 总权重 × 40

考试权重：期末=2 | 期中=1.5 | 月考/单元测=1

示例：某学期有 期中(85/100) + 期末(90/100) + 月考(80/100)
  得分率加权平均 = (0.85×1.5 + 0.90×2 + 0.80×1) / (1.5+2+1) = 3.875 / 4.5 = 0.861
  学业成长分 = 0.861 × 40 = 34.4 → 34 分
  学业显示分 = 基础分(如60) + 34 = 94 分
```

### 2.3 心理维度：关注自己，系统就关注你

| 行为 | 影响类型 | 成长分变化 | 说明 |
|------|----------|-----------|------|
| **记录心情日记** | 直接加分 | +0.3/次，封顶12 | 每学期记录越多，心理分越高 |
| **连续记录7天** | 额外奖励 | +2 | 连续性奖励 |
| **连续记录14天** | 额外奖励 | +4 | 更大奖励 |
| **连续记录30天** | 额外奖励 | +6 | 封顶奖励 |
| **完成心理测评** | 直接加分 | +6~12 | MHT/SCL90 等标准化测评 |
| **主动心理咨询** | 大幅加分 | +4 | 主动求助是积极行为 |
| **教师评语"情绪稳定"** | 间接加分 | +2 | 评语标签影响 |
| **30天无心情记录** | 衰减惩罚 | ×0.85 | 系统会预警并提醒 |
| **情绪波动期** | 自然下降 | -3~5 | 正常波动，不需要焦虑 |

**心理维度计算公式：**
```
心理成长分 = 测评得分(0~12) + 心情记录(0~12) + 连续奖励(0~6) + 主动求助(0~4) + 参与度(0~6)

示例：某学生
  - 完成 MHT 测评得分 80/100 → 测评得分 = 80/100 × 12 = 9.6 → 10
  - 本学期记录心情 20 次 → 心情记录 = 9 (14次档位)
  - 连续记录 15 天 → 连续奖励 = 4
  - 主动心理咨询 1 次 → +4
  - 参与度 = 6
  心理成长分 = 10 + 9 + 4 + 4 + 6 = 33 → 封顶 40 = 33 分
  心理显示分 = 基础分60 + 33 = 93 分 (优秀)
```

### 2.4 职业维度：越早探索，越早清晰

|------|----------|-----------|------|

| **完成MBTI测评** | 激活加分 | +8 | 两种测评都完成，职业维度快速启动 |

| **大学参观/职业体验** | 直接加分 | +3 | 深度探索 |

```

  - 参与度 = 8
  职业成长分 = 16 + 11 + 8 = 35 → 封顶 40 = 35 分
  职业显示分 = 基础分60 + 35 = 95 分 (优秀)
```

| 行为 | 影响类型 | 成长分变化 | 说明 |
|------|----------|-----------|------|
| **参加一次集体活动** | 直接加分 | +3 | 活动参与是核心加分项 |

**社交维度计算公式：**
```
社交成长分 = 活动参与(0~16) + 教师评语(0~12) + 角色担当(0~12)

活动参与得分：
  - 1-2 次活动 → +3~6
  - 3-4 次活动 → +9~12

  - 每有一条正向社交标签 → +2
  - 封顶 12 分

角色担当：
  - 普通成员 → +4
  - 小组长/干事 → +8
  - 班长/社长 → 封顶 12

示例：某学生
  - 本学期参加活动 4 次 → 活动参与 = 12
  - 教师评语有 3 条正向社交标签 → 教师评语 = 6
  - 担任社团干事 → 角色担当 = 8
  社交成长分 = 12 + 6 + 8 = 26 → 封顶 40 = 26 分
  社交显示分 = 基础分60 + 26 = 86 分 (优秀)
```

### 2.6 特长维度：不练真的会退步

| 行为 | 影响类型 | 成长分变化 | 说明 |
|------|----------|-----------|------|
| **里程碑申报通过** | 直接加分 | +4/个 | 技能考级、比赛获奖、作品发表 |
| **比赛获奖** | 跨维加分 | 特长+3~6, 学业/社交+1~2 | 荣誉有溢出效应 |
| **创作作品** | 直接加分 | +3~5 | 原创作品、发明创造 |
| **持续训练/出勤** | 间接加分 | +1~3 | 通过参与度体现 |
| **特长完全不练** | 大幅衰减 | ×0.7 | 衰减最快！不练真的会退步 |
| **减少练习时间** | 自然回落 | -2~5 | 成长分逐渐下降 |

**特长维度计算公式：**
```
特长成长分 = 里程碑(0~16) + 成果产出(0~12) + 参与度(0~12)

里程碑得分：
  - 1 个里程碑 → +4
  - 2 个里程碑 → +8
  - 3 个里程碑 → +12
  - 4+ 个里程碑 → 封顶 16

成果产出：
  - 校级奖项 → +3
  - 区级奖项 → +6
  - 市级及以上 → +9~12

参与度：
  - 每周训练 1-2 次 → +6
  - 每周训练 3+ 次 → 封顶 12

示例：某学生
  - 钢琴七级通过 + 原创作品发表 → 里程碑 = 8
  - 校园音乐节演出获奖 → 成果产出 = 6
  - 每周训练 3 次 → 参与度 = 12
  特长成长分 = 8 + 6 + 12 = 26 分
  特长显示分 = 基础分60 + 26 = 86 分 (优秀)
```

## 四、衰减机制：长期不做，真的会掉

### 4.1 衰减规则

如果某维度**连续两个学期无有效数据输入**，成长分将按以下系数衰减：

| 维度 | 衰减系数 | 含义 | 示例 |
|------|----------|------|------|
| 学业 | ×0.90 | 小幅衰减 | 上学期成长分30 → 本学期27 |
| 心理 | ×0.85 | 中等衰减 | 上学期成长分20 → 本学期17 |
| 职业 | ×0.95 | 几乎不衰减 | 上学期成长分15 → 本学期14 |
| 社交 | ×0.80 | 明显衰减 | 上学期成长分20 → 本学期16 |
| 特长 | ×0.70 | 大幅衰减 | 上学期成长分20 → 本学期14 |

**注意**：衰减只影响**成长分**，不影响**基础分**。基础分继续按规则增长。

这意味着：
- 如果基础分还在涨，成长分小幅衰减 → 显示分可能**持平或微降**
- 如果基础分已封顶，成长分衰减 → 显示分**明显下降**

### 4.2 衰减示例

```
学生小李：K9春 → K10秋，完全放弃特长训练

K9 春：特长成长分 = 22，基础分 = 60，显示分 = 82
K10 秋：无里程碑、无训练、无成果
  → 特长成长分 = 22 × 0.7 = 15.4 → 15
  → 特长显示分 = 60 + 15 = 75

变化：82 → 75，下降了 7 分！
系统提示："你本学期特长活动较少，建议重新开始训练"

如果 K10 春继续不练：
  → 特长成长分 = 15 × 0.7 = 10.5 → 11
  → 特长显示分 = 60 + 11 = 71 (标签从"优秀"降为"良好")
```

## 五、系统如何确保全面发展

### 5.1 问题：学生只刷一个维度怎么办？

有些学生可能发现"只要拼命考试，学业维度就会很高"，于是只关注成绩，忽视其他方面。

**系统的对策**：

#### 对策 1：基础分封顶 + 成长分上限

基础分最多 60 分，成长分最多 40 分。**即使学业成长分刷满 40，学业显示分也只有 100，无法无限膨胀。**

更重要的是：**一个维度的满分不代表全面发展。**

#### 对策 2：五维均衡度评价

系统计算五维的**标准差**：

```
均衡度 = 100 - 五维标准差 × 2

示例 A：五维 = [92, 88, 80, 84, 82]
  均值 = 85.2, 标准差 = 4.2
  均衡度 = 100 - 4.2 × 2 = 91.6 (非常均衡)

示例 B：五维 = [96, 72, 68, 66, 65]
  均值 = 73.4, 标准差 = 13.1
  均衡度 = 100 - 13.1 × 2 = 73.8 (不均衡)
```

均衡度在学期档案中显示，引导学生关注短板。

#### 对策 3：待办清单智能推荐

系统根据五维最低维度，智能推荐待办事项：

| 最低维度 | 推荐待办 | 预期提升 |
|----------|----------|----------|
| 心理最低 | 「记录本周心情」→「做一次心理测评」 | +3~5 |
| 社交最低 | 「参加一次社团活动」→「报名志愿者」 | +4~6 |
| 特长最低 | 「申报一个里程碑」→「开始一项兴趣训练」 | +4~6 |
| 职业最低 | 「完成霍兰德测评」→「设定一个学期目标」 | +5~8 |
| 学业最低 | 「整理错题本」→「找老师答疑」 | +3~5 |

#### 对策 4：评优挂钩

学校的评优评奖（如三好学生、优秀毕业生）可以设置**五维门槛**：

| 奖项 | 五维要求 |
|------|----------|
| 三好学生 | 无维度低于 65 |
| 优秀毕业生 | 无维度低于 70，均衡度 ≥ 85 |
| 全能之星 | 五维均值 ≥ 85，均衡度 ≥ 90 |

这从制度层面强制学生关注全面发展。

#### 对策 5：雷达图视觉引导

雷达图的不规则形状本身就是一种视觉反馈：

```
均衡的雷达图（圆润）          不均衡的雷达图（尖刺）
                              
        学业                      学业
       /    \                     |
      /  ★   \                   / \
 特长 ──┼─── 心理              特长    心理
      \      /                   \   /
       \    /                     \ /
        社交                       社交
        职业                       职业
        
"全面发展，五维开花"          "一枝独秀，需要补弱"
```

### 5.2 全面发展典型案例

## 六、插班生全面发展保障

### 6.1 插班生的特殊挑战

插班生面临的全面发展问题更严重：
- 入学时基础分低（40 vs 在校生的 57）
- 没有历史数据，雷达图"从零开始"
- 不了解学校体系，不知道该怎么参与

### 6.2 系统的针对性保障

| 机制 | 如何保障全面发展 |
|------|-----------------|
| **观察期加速** | 基础分 +5/学期（普通生+3.5），快速缩小差距 |
| **新手保护** | 各维度成长分保底 10-15，避免初始分数打击信心 |
| **行为替代数据** | 一次心情记录 = 心理维度参与，降低参与门槛 |
| **专属待办清单** | 系统推荐「完成测评→参加活动→记录心情」，引导全面参与 |
| **同伴对比隐藏** | 不显示班级平均，只和自己比，减少焦虑 |

## 十、FAQ

**Q1：如果我什么都不做，五维会掉吗？**
> 基础分不会掉（只要在校就会涨），但成长分会因衰减机制下降。长期什么都不做，某些维度会明显落后。

**Q2：我只关心成绩，其他维度不重要吧？**
> 学业维度最多占 100 分中的 40 分成长分。如果你只关注成绩，最多只能让学业显示分达到 100，但其他维度会很低。评优时均衡度不达标，可能错失奖项。

**Q3：插班生永远追不上在校生吗？**
> 不会。观察期后基础分增速相同，成长分完全取决于个人表现。张新航的案例显示，K10 春就已经超越部分在校生了。

**Q4：心理维度记录心情会不会暴露隐私？**
> 心情记录只显示「记录次数」和「情绪趋势」，具体内容和心理测评详情只有心理老师可见，家长看到的是摘要。

**Q5：我可以刷分吗？**
> 系统设计为「行为即记录」，你无法在没有真实行为的情况下刷分。而且成长分有封顶，刷满一个维度对你的整体发展帮助有限。

**Q6：我的五维下降了怎么办？**
> 首先不要焦虑，查看是哪个维度下降了，系统会推荐具体的提升行为。通常只需要做一两件小事（参加活动、记录心情、设定目标），下个学期就能回升。
