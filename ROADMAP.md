# 新府学「一生一案」—— 长线优化与拓展路线图

> 参考系统：Duolingo（游戏化/连续打卡）、Strava（数据可视化/社交比较）、Notion（模板化/自由度）、ClassDojo（教育行为积分）、Headspace（心情追踪/进度可视化）、Habitica（RPG游戏化）、LinkedIn（职业档案/技能认可）

---

## 核心设计原则

**"看见自己 → 被看见 → 主动经营"**

```
学生打开系统 → 一眼看到"我这周进步了吗" → 发现还可以做什么 → 行动 → 获得反馈 → 循环
```

## Phase 1：游戏化激励体系（用户留存核心）

### 1.1 连续打卡体系（参考 Duolingo / Headspace）

| 打卡项 | 周期 | 奖励 | 视觉设计 |
|--------|------|------|----------|
| **心情日记** | 每日 | 连续7天+10积分，30天+50积分 | 日历热力图，连续天数火焰图标 |
| **五维自评** | 每周 | 完成周反思+20积分 | 周打卡环，未完成显示灰色缺口 |
| **里程碑记录** | 随时 | 每个+5~20积分 | 成就弹窗动画 |
| **活动参与** | 每次 | 按活动类型+积分 | 活动类型专属徽章 |

**关键设计**：
- 首页顶部常驻"本周打卡进度"横向进度条
- 断签后显示"就差1天，连续X天的记录"挽回提示
- 教师端可看到班级打卡率，作为班级氛围参考

### 1.2 成就徽章系统（参考 Strava / Duolingo）

```typescript
// 徽章分类
const BADGE_CATEGORIES = {
  学业: ["月考进步", "学科均衡", "全A战士", "逆袭之星"],
  心理: ["心情达人", "连续记录30天", "积极心态", "压力管理师"],
  职业: ["职业探索者", "目标设定师", "路径规划者"],
  社交: ["团队之星", "沟通达人", "领导力徽章"],
  特长: ["社团骨干", "竞赛获奖者", "才艺展示"],
  综合: ["五维全优", "成长之星", "学期全勤"],
};
```

**展示位置**：
- 学生个人档案页顶部「荣誉墙」
- 成长空间解锁条件与徽章挂钩
- 获得徽章时全屏动画+音效（可关闭）

### 1.3 等级成长体系（参考 Habitica）

```
Lv.1 新生 → Lv.5 探索者 → Lv.10 成长者 → Lv.20 领航者 → Lv.50 卓越者
```

**升级公式**：经验值 = 积分累积 + 连续打卡奖励 + 里程碑加成 + 教师评语质量分

**等级特权**：
- 每升5级解锁成长空间新区域
- 高等级学生可申请「学长导师」身份
- 等级影响学期档案中的评价维度

## Phase 3：社交与反馈层（让用户"被看见"）

### 3.1 教师评语互动优化

**当前**：学生被动接收评语
**优化**：
- 学生可「收藏」特别受触动的评语
- 学生可「感谢」教师（发送匿名感谢卡片）
- 评语显示「已读」状态给教师
- 家长端同步收到评语通知

### 3.2 班级里程碑墙（参考 Instagram Feed）


- 班主任可置顶重要里程碑

### 3.3 同伴成长对比（参考 Strava 好友对比）

## Phase 4：AI 自然赋能（不是"AI功能"而是"AI助手"）

### 4.1 AI 评语助手 → 智能草稿（已完成基础）

**优化方向**：
- 教师输入3个关键词 → AI生成3个版本供选择 → 教师修改 → 保存
- AI自动记录教师的修改偏好，下次生成更接近教师风格
- 批量模式：选择5个学生 → AI为每个学生生成个性化评语草稿

### 4.2 AI 心理预警智能分析

**当前**：展示预警列表
**优化**：
```
⚠️ 学生：张明
预警类型：学习焦虑测评 - 高分（24/30）

AI分析：
• 该生近30天心情记录平均2.3分（偏低）
• 最近2次数学测验成绩下降15分
• 建议关注：考试压力 + 数学学习困难

推荐行动：
□ 与心理老师预约面谈
□ 与数学教师沟通调整学习计划
□ 家长沟通（生成家长沟通报告草稿）
□ 标记为持续关注（每周自动复查）
```

### 4.3 AI 学期档案自动生成

**触发条件**：学期末自动触发
**生成内容**：
- 学期成绩总览（含趋势分析）
- 五维雷达变化解读
- 里程碑亮点回顾
- 教师评语精选
- AI个性化成长建议
- 下学期目标建议

**格式**：自动生成图文并茂的PDF档案 + 网页版互动档案

### 4.4 AI 成长助手（学生端聊天机器人）

```
🤖 小星（AI成长助手）

学生：我这学期进步了吗？
小星：你这学期相比上学期，五维雷达面积扩大了23%！
      尤其在「社交」维度进步最大（+15分）。
      建议：可以多参加社团活动，进一步发挥优势。
      需要我帮你设定下学期的目标吗？

学生：数学最近有点难
小星：我看到你最近3次数学成绩是 85→78→72。
      建议：
      1. 和数学老师约个时间讨论薄弱环节
      2. 参加周三的数学兴趣小组
      3. 试试调整学习方法（推荐：费曼学习法）
      需要我把这些建议整理成行动计划吗？
```

## Phase 6：技术架构升级

### 6.1 性能优化

| 优化项 | 当前状态 | 目标 |
|--------|----------|------|
| 数据库查询 | 无缓存 | Redis缓存热点数据 |
| 图片存储 | 本地文件 | MinIO对象存储 |
| 静态资源 | 无CDN | Nginx/Cloudflare CDN |
| 大数据报表 | 实时查询 | 定时任务预聚合 |
| API响应 | 无分页 | 统一分页+游标分页 |

### 6.2 移动端适配

- PWA渐进式Web应用（可安装到手机桌面）
- 响应式布局优化（当前已基础适配）
- 移动端专属交互（下拉刷新、左滑操作）
- 推送通知（心情提醒、预警通知）

### 6.3 数据安全加固

- 操作审计日志（谁在什么时间做了什么）
- 敏感数据加密（心理测评详情）
- 数据导出权限控制
- 定期安全扫描

## 成功指标（OKR）

### 学生端
- **O**：学生主动使用系统管理自己的成长
- KR1：日活跃学生占比 > 60%
- KR2：心情日记连续7天打卡率 > 40%
- KR3：学期档案查看率 > 90%

### 教师端
- **O**：教师高效记录学生成长，AI辅助减少负担
- KR1：评语平均撰写时间 < 5分钟（当前约15分钟）
- KR2：AI评语采用率 > 50%
- KR3：心理预警响应时间 < 48小时

### 管理端
- **O**：数据驱动学校管理决策
- KR1：管理端周报查阅率 > 80%
- KR2：数据报表使用频次 > 每周2次
- KR3：家长满意度 > 85%
